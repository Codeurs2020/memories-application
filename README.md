## Installation Guide

### Production

- Clone this project

```
git clone https://github.com/tinspham209/memories-application
```

- cd to `/server`

- Install dependencies

```
npm install
```

- Start the server

```
npm start
```

- cd to `/client`

- Install dependencies

```
npm install
```

- Start the server

```
npm start
```

### Development

- Clone this project
- cd to both: `/client` & `/server`
- Install dependencies

```
npm install
```

- Start packager

```
npm start
```
